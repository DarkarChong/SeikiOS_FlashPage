About

SEIKIOS is a non-profit Flash game preservation project. We manually decompile broken SWF files using JPEXS Free Flash Decompiler, patch the broken AS2 code, and publish the playable versions running via Ruffle — an open-source Flash emulator. No Flash plugin, no installation, no tracking.

Rights & Takedowns

All games hosted on SEIKIOS belong to their original creators. We host patched versions solely for cultural preservation, non-commercially, and only games that we believe are effectively abandoned by their original distributors.

If you are a rights holder and want a game removed, email chong.seikin@gmail.com with the game's URL and proof of ownership. We comply with valid takedown requests within 48 hours. You may also open an issue on our GitHub repository.
