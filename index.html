About

SEIKIOS is a non-profit Flash game preservation project. We manually decompile broken SWF files using JPEXS Free Flash Decompiler, fix AS2 code bugs, and publish the patched games running via Ruffle — an open-source Flash emulator.

All rights belong to their original creators. If you are a rights holder and want a game removed, please open a GitHub issue.
